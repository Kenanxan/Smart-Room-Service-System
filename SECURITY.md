# Security Notes

This repository is intentionally scoped as a public-safe sample.

## Allowed Content

- static frontend previews
- mock data
- generic UI examples
- documentation that describes the product at a high level

## Not Allowed

- backend code
- secrets, tokens, API keys, or certificates
- `.env` files
- real customer or business data
- production endpoints
- deployment credentials
- internal admin workflows tied to private operations

## Publishing Checklist

- verify no backend files were copied
- verify no environment files exist
- verify no real domains or base URLs remain
- verify no authentication code remains
- verify all examples run on mock data only
